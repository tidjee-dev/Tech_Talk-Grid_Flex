  - `row` (par defaut)

- `justify-content`

  - `stretch` (par defaut)
  - `flex-start`
  - `center`
  - `flex-end`
  - `space-between`
  - `space-around`

- `align-items`

  - `stretch` (par defaut)
  - `flex-start`
  - `center`
  - `flex-end`
  - `baseline`

- `flex-wrap`

  - `nowrap` (par defaut)
  - `wrap`
  - `wrap-reverse`

- `align-content`

  - `stretch` (par defaut)
  - `flex-start`
  - `center`
  - `flex-end`
  - `space-between`
  - `space-around`

- `order`

  - `auto` (par defaut)
  - Valeur numérique pour modifier l'ordre d'apparition

- `flex-grow`

  - `auto` (par defaut)
  - Valeur numérique pour modifier la proportion de l'espace disponible

- `flex-shrink`

  - `auto` (par defaut)
  - Valeur numérique pour modifier la proportion de l'espace disponible

- `flex-basis`

  - `auto` (par defaut)
  - Valeur numérique pour modifier la proportion de l'espace disponible
