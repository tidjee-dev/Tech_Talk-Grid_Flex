Flexbox vs. Grid

Découvrez les différences entre Flexbox et Grid, leurs avantages, leurs inconvénients, et quand les utiliser dans vos projets web.

Flexbox

Usage : Alignement en une dimension (ligne ou colonne).
Avantages : Simple, rapide, idéal pour des composants flexibles.
Inconvénients : Limité à un seul axe, pas optimal pour des layouts complexes.
Propriétés clés : flex-direction, justify-content, align-items.

Grid

Usage : Layout en deux dimensions (ligne et colonne).
Avantages : Contrôle total, idéal pour des mises en page complexes.
Inconvénients : Plus complexe à prendre en main, nécessite de définir une structure rigide à l'avance.
Propriétés clés : grid-template-rows, grid-template-columns, gap.

Flexbox

Pour définir un conteneur Flexbox, utilisez la propriété display: flex.

  • Propriétés sur le conteneur
    • flex-direction: Définit la direction des éléments enfants (lignes ou colonnes).
    • justify-content: Alignement des enfants sur l'axe principal 1.
    • align-items: Alignement des enfants sur l'axe secondaire 1.
    • flex-wrap: Gestion du retour de ligne.
    • gap: Espacement entre les enfants.
  • Propriétés sur les enfants
    • order: Change l'ordre des enfants.
    • flex-grow: Augmentation de la taille de l'enfant par rapport aux autres enfants.
    • flex-shrink: Diminution de la taille de l'enfant par rapport aux autres enfants.
    • flex-basis: Taille initiale de l'enfant.
    • align-self: Alignement vertical de l'enfant.

1 L'axe principal est défini par la propriété flex-direction.
Par défaut, avec flex-direction: row, l'axe principal est horizontal, et l'axe secondaire est vertical.
Inversement, avec flex-direction: column, l'axe principal devient vertical et l'axe secondaire est horizontal.

Grid

Pour définir un conteneur Grid, utilisez la propriété display: grid.

  • Propriétés sur le conteneur
    • grid-template-columns: Définit la largeur des colonnes. 1
    • grid-template-rows: Définit la hauteur des lignes. 1
    • grid-template-areas: Définit les zones de la grille. 1 3
    • grid-gap: Espacement entre les enfants.
    • grid-column-gap: Espacement entre les colonnes.
    • grid-row-gap: Espacement entre les lignes.
    • grid-auto-flow: Gestion du retour de ligne.
    • justify-items: Alignement horizontal des enfants.
    • align-items: Alignement vertical des enfants.
    • justify-content: Alignement horizontal des colonnes.
    • align-content: Alignement vertical des lignes.
  • Propriétés sur les enfants
    • grid-area: Zone de la grille occupée par l'enfant. 3
    • grid-column: Colonne(s) de la grille occupée(s) par l'enfant. 4
    • grid-row: Ligne(s) de la grille occupée(s) par l'enfant. 4
    • justify-self: Alignement horizontal de l'enfant.
    • align-self: Alignement vertical de l'enfant.
    • order: Change l'ordre des enfants.

1 Dans un conteneur CSS Grid, l'unité fr représente une fraction de l'espace disponible. Cela permet de répartir automatiquement l'espace en proportions flexibles et réactives. Il est néanmoins possible de combiner plusieurs unités (fr, %, px, ...) pour obtenir des proportions plus complexes.

2 repeat() est une fonction qui permet de répeter des valeurs de dimension. Par exemple, repeat(3, 1fr) représente trois colonnes de largeur flexible.

3 grid-area est une propriété qui permet de placer un enfant dans une zone particulière de la grille. La valeur de grid-area doit correspondre au nom d'une zone définie dans la propriété grid-template-areas du conteneur.
Ex : 

/* Conteneur */
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 1fr auto 1fr;
    grid-template-areas:
        "header header header"
        "nav main sidebar"
        "footer footer footer";
}

/* Enfants */
.header {
    grid-area: header;
    background-color: lightblue;
}
.nav {
    grid-area: nav;
    background-color: lightcoral;
}
.main {
    grid-area: main;
    background-color: lightgreen;
}
.sidebar {
    grid-area: sidebar;
    background-color: lightpink;
}
.footer {
    grid-area: footer;
    background-color: lightskyblue;
}

4 grid-column et grid-row sont des propriétés qui permettent de placer un enfant dans une ligne ou une colonne particulière de la grille.
La valeur de grid-column ou grid-row doit correspondre aux indices des lignes ou des colonnes de la grille.
Si la colonne ou la ligne occupe plusieurs cellules, il faut utiliser span pour indiquer le nombre de cases qu'elle occupe.
Ex : 

<!-- HTML -->

<div class="container">
    <div class="child child-2">1</div>
    <div class="child">2</div>
    <div class="child">3</div>
    <div class="child">4</div>
    <div class="child">5</div>
    <div class="child">6</div>
</div>



/* CSS */

/* Conteneur */
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(3, 1fr);
    gap: 5px;
}

/* Enfants */
.child {
    background-color: lightgray;
    border: 1px solid black;
    display: flex;
    justify-content: center;
    align-items: center;
    font-size: 24px;
}
.child-2 {
    background-color: coral;
    grid-column: 2 / span 2;
    grid-row: 2 / span 2;
}
